# 校园二手交易平台 v2.0 — 项目总结

## 项目概况

校园二手交易平台是一个基于 Spring Boot 3 + Vue 3 的全栈 Web 应用，支持商品发布、浏览、搜索、议价、交易、收藏、即时通讯、管理后台等功能。

**v2.0 改造时间**：2026-07-14 ~ 2026-07-15

### v1.0 → v2.0 核心变化

| 维度 | v1.0 改造前 | v2.0 改造后 |
|------|------------|------------|
| 认证 | 无（token = userId 字符串） | JWT（Access 2h + Refresh 7d + 黑名单） |
| 密码 | MD5 | BCrypt + 老用户自动升级 |
| 权限 | 手动 isAdmin() 判字符串 | Spring Security @PreAuthorize |
| 缓存 | 无 | Redis（收藏 Set + 浏览量计数器 + 限流计数器） |
| 搜索 | MySQL LIKE | ES 预留 + MySQL 降级（ProductSearchService 解耦） |
| 消息 | HTTP 轮询 | WebSocket + STOMP 实时推送 + 通知落库 |
| 订单 | 基础 CRUD + 手动状态改 | 完整状态机 + TaskScheduler 30min 自动取消 |
| 评价 | 仅买家评卖家 | 双向互评 + 评分统计 |
| 图片 | 存原图 | 上传自动生成 200px 缩略图 |
| 管理后台 | 无 | ECharts 统计大屏 + AOP 操作日志 |
| 文档 | 无 | Knife4j Swagger（/doc.html） |
| 部署 | 手动打 jar 上传 | Docker Compose 一键启动 + GitHub Actions CI |
| 安全加固 | 含越权漏洞 | 接口限流 + 举报系统 + 完整权限体系 |

## 功能模块（完整清单）

### 用户模块
- 注册/登录（BCrypt 加密，老用户 MD5 自动升级）
- JWT 双 Token（Access 2h + Refresh 7d + 黑名单）
- 个人信息管理（昵称/头像/手机/学校）
- 个人主页统计（发布数/卖出/买入）
- 接口限流（登录 10次/分钟，注册 5次/分钟）

### 商品模块
- 发布/编辑/下架商品
- 分类浏览（6 个分类）+ 多条件筛选（成色/价格）
- MySQL LIKE 搜索（ES 预留，接口已解耦）
- 相似商品推荐（同分类 + 价格 ±30%）
- 浏览量统计（Redis 计数 + 定时刷库）
- 图片上传（缩略图自动生成 200px）

### 订单模块
- 下单（乐观锁防超卖 + 生成订单号）
- 支付（模拟，状态 0→1）
- 发货（卖家，1→2）
- 确认收货（买家，2→3）
- 手动取消（买家，0→4）
- TaskScheduler 自动取消（30 分钟未支付）

### 收藏模块
- Redis Set + MySQL 双写
- SISMEMBER 优先读
- 收藏/取消收藏

### 评价模块
- 双向互评（买家⇄卖家）
- 1-5 星评分
- 好评率计算
- 评分自动更新 User 表

### 议价模块
- 买家出价（防重复 + 防自己商品）
- 卖家接受/拒绝（权限校验）
- 出价列表（买家视角 + 卖家视角）

### 消息模块
- WebSocket + STOMP 实时推送
- 离线消息 MySQL 存储
- 会话列表 + 未读计数
- 消息标记已读
- 发消息时自动创建通知

### 通知模块
- 通知落库（NEW_MESSAGE 等类型）
- 通知列表/未读计数/标记已读
- 消息发送时自动创建通知

### 举报模块
- 提交举报（指定类型/原因/描述）
- 管理员审核列表
- 处理（通过/驳回）

### 管理后台
- 用户管理（列表/删除）
- 商品管理（列表/删除/状态修改）
- 公告管理（CRUD，admin 权限）
- 数据统计大屏：概览卡片/用户趋势/订单趋势/分类占比/热门商品 TOP10
- 操作日志（AOP + t_operation_log）
- 举报审核

### AI 助手
- DeepSeek API 集成
- 商品购买建议/价格评估

## 安全架构

```
请求 → JwtAuthFilter(验签+黑名单) → SecurityConfig(URL规则) → @PreAuthorize(方法级) → Controller
```

- 公开接口：登录/注册、商品浏览、文件下载、公告查看、Swagger 文档
- 认证接口：发布商品、下单、聊天、收藏、评价、出价、举报
- 管理员接口：管理后台、统计大屏、公告管理、举报审核

## 本地运行

### 方式一：双击 start.bat 一键启动（推荐）
```bash
# 双击项目根目录下的 start.bat
# 自动执行：检查启动 Redis → 检查启动 MySQL → 启动后端 → 打开浏览器
# 前提：Maven 已安装并加入 PATH
```

### 方式二：手动启动
```bash
# 前置条件：MySQL 8.0 + Redis 7 已启动
# 导入数据库
mysql -u root -p123456 campus_trade < init.sql
mysql -u root -p123456 campus_trade < campus-trade-server/src/main/resources/db/migration/V1__upgrade.sql
mysql -u root -p123456 campus_trade < campus-trade-server/src/main/resources/db/migration/V2__notifications.sql
mysql -u root -p123456 campus_trade < campus-trade-server/src/main/resources/db/migration/V3__offer_report.sql

# 启动后端
cd campus-trade-server && mvn spring-boot:run
```

### 方式三：Docker Compose 一键启动
```bash
docker compose up -d
# 访问 http://localhost:8080
```
